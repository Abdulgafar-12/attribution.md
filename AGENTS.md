# AGENTS.md

This is a specification repository for the AI Attribution Protocol. It is not a software project.

## Guidelines

- Do not modify normative specification language (MUST, SHOULD, MAY) without explicit instruction.
- Do not generate code implementations. The spec contains pseudocode for illustration only.
- Changes to `README.md` affect the published specification. Treat edits with the same care as an RFC.
- The `index.html` file is the public landing page served at https://attribution.md.
- The `schema.json` file must remain consistent with the specification in `README.md`.

## Attribution

This repository participates in the AI Attribution Protocol. See ATTRIBUTION.md for reciprocity guidelines.
